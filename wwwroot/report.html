HỆ THỐNG THƯƠNG MẠI ĐIỆN TỬ MODELSTORE PLATFORM

Ngày thực hiện: 08/05/2026

1. Tổng Quan Dự Án

ModelStore là một nền tảng thương mại điện tử chuyên biệt dành cho các sản phẩm mô hình (PVC, Resin). Mục tiêu của dự án là chuyển đổi từ kiến trúc Microservices cũ sang kiến trúc Monolithic (Đơn khối) mạnh mẽ, dễ bảo trì và hiệu suất cao sử dụng công nghệ ASP.NET Core.

2. Công Nghệ Sử Dụng (Technology Stack)

Dự án được xây dựng trên những công nghệ hiện đại nhất hiện nay:

  • Backend: C# (.NET 9), ASP.NET Core Web API, EF Core.
  • Database: MySQL (Quản lý dữ liệu tập trung).
  • Frontend: HTML5, CSS3 hiện đại, Vanilla Javascript (ES6+).
  • Thống kê: Chart.js (Biểu đồ doanh thu và đơn hàng).
  • Tài liệu: Swagger UI (Giao diện kiểm thử API trực quan).

3. Quy Trình Thực Hiện

  • Giai đoạn 1: Thiết lập nền tảng, cấu hình kết nối Database và định nghĩa các thực thể (Models).
  • Giai đoạn 2: Xây dựng hệ thống API cho Sản phẩm, Đơn hàng, Người dùng và cấu hình hệ thống.
  • Giai đoạn 3: Thiết kế giao diện Premium Dark Mode, tích hợp gọi API và xử lý giỏ hàng real-time.
  • Giai đoạn 4: Xây dựng Dashboard quản trị, biểu đồ thống kê và tính năng xuất báo cáo.
  • Giai đoạn 5: Tích hợp thanh toán trực tuyến qua cổng VNPay.

4. Các Tính Năng Then Chốt

  • Đặt hàng chuyên nghiệp: Quy trình từ xem hàng, thêm giỏ đến thanh toán được tối ưu hóa.
  • Quản trị thông minh: Dashboard hiển thị biểu đồ doanh thu theo ngày và trạng thái đơn hàng.
  • Xuất báo cáo Word: Hỗ trợ Admin xuất danh sách đơn hàng và người dùng ra file .doc nhanh chóng.
  • Cấu hình động: Thay đổi nội dung Menu và Danh mục trang chủ mà không cần can thiệp mã nguồn.

5. Kết Luận

Dự án ModelStore đã hoàn thiện toàn bộ các tính năng từ cơ bản đến nâng cao. Hệ thống hoạt động ổn định, giao diện sang trọng và có khả năng mở rộng cao trong tương lai.
