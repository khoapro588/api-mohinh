# 🌟 ModelStore - Premium Figures & Models E-Commerce

**ModelStore** là một nền tảng thương mại điện tử hiện đại chuyên cung cấp các dòng mô hình tĩnh cao cấp (Gundam, PVC Figure, Resin Statue...). Dự án được xây dựng với mục tiêu mang lại trải nghiệm mua sắm mượt mà, đẳng cấp cho cộng đồng sưu tập mô hình tại Việt Nam.

---

## 🚀 Tính năng nổi bật

- **🤖 AI Chatbot Assistant:** Tích hợp Google Gemini AI để tư vấn sản phẩm, giải đáp thắc mắc và gợi ý mô hình phù hợp với sở thích người dùng.
- **💳 Thanh toán VNPay:** Tích hợp cổng thanh toán điện tử VNPay, đảm bảo giao dịch nhanh chóng và an toàn.
- **📦 Quản lý giỏ hàng & Đơn hàng:** Hệ thống giỏ hàng thông minh, quản lý trạng thái đơn hàng thời gian thực.
- **✨ Giao diện Premium:** Thiết kế Dark Mode sang trọng, hiệu ứng mượt mà, tối ưu hóa trải nghiệm người dùng (UX/UI).
- **🛠 Admin Dashboard:** Hệ thống quản trị mạnh mẽ cho phép quản lý sản phẩm, đơn hàng và cài đặt hệ thống dễ dàng.

## 💻 Công nghệ sử dụng

- **Backend:** ASP.NET Core 9.0 Web API
- **Database:** MySQL / Entity Framework Core
- **Frontend:** HTML5, Vanilla CSS3, Javascript (ES6+)
- **AI Integration:** Google Generative AI (Gemini 2.5 Flash)
- **Payment Gateway:** VNPay API
- **Auth:** Custom JWT Authentication & Middleware

## 🛠 Hướng dẫn cài đặt

1. **Clone dự án:**
   ```bash
   git clone https://github.com/khoapro588/api-mohinh.git
   ```

2. **Cấu hình Database:**
   Cập nhật chuỗi kết nối trong `appsettings.json` phù hợp với MySQL của bạn.

3. **Chạy Migration:**
   ```bash
   dotnet ef database update
   ```

4. **Khởi chạy ứng dụng:**
   ```bash
   dotnet run
   ```
   Truy cập `http://localhost:5271` để trải nghiệm.
